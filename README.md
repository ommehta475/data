# School Accounting Analytics Dashboard

A single-page, beautiful dashboard for **school accounting data analysis**. It shows:

- Key KPIs (total fees billed, collected, outstanding, net balance)
- Monthly **income vs expenses** line chart
- **Fee collection by class** bar chart
- A searchable table of **student-wise outstanding dues**

Built with plain **HTML, CSS, and JavaScript** plus [Chart.js](https://www.chartjs.org/) from CDN, so you can open it directly in any modern browser.

## Files

- `index.html` – main page layout and structure
- `styles.css` – modern dark theme UI styling
- `script.js` – sample accounting data, KPIs, charts, and interactivity

## How to Run

1. Place all three files (`index.html`, `styles.css`, `script.js`) in the same folder.
2. Double‑click `index.html` (or right‑click → **Open with** → your browser).
3. Use the filters at the top to switch session and class; use the search box to find students.

You can replace the **sample data** in `script.js` with your real school accounting data (sessions, class-wise fees, and student dues).
